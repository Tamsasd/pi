# Pi

> A simple PI digit memorization contest page for pi day 2026 (03.14.)

## Features

- **Lives:** Only fail an attempt after entering 3 incorrect digits
- **Cheat prevention:**
  - Automatically fail attempt upon leaving the page
  - Only allow one name-class pair
  - Login system so only authorized devices can make changes to the database
- **Leaderboard:** Seperate ranking page for the students to see the top-list and scores in real-time, without having access to a logged-in device

## Tech Stack

- **Language:** Javascript
- **Frontend:** HTML, CSS
- **Backend/Database:** Firebase
